# Sistema de Adoção de Pets

Projeto desenvolvido para a disciplina de Projeto de Programação.

O sistema tem como objetivo auxiliar uma ONG no gerenciamento do processo de adoção de animais, permitindo o cadastro de pets, adotantes e o controle das regras de negócio relacionadas ao processo de adoção.

## Tecnologias Utilizadas

* Java 21
* Maven
* JUnit 5
* GitHub Actions

## Como Executar

Clone o repositório:

```bash
git clone <url-do-repositorio>
```

Acesse a pasta do projeto:

cd pet-adocao

Compile o projeto:


mvn compile


## Executar os Testes

Para executar  os testes unitários:


mvn test(no o powershell)


## Integração Contínua(gihub actions)

O projeto utiliza GitHub Actions para executar automaticamente o processo de build e os testes unitários a cada push ou pull request para a branch principal.
